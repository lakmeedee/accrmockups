az managedcleanroom collaboration create
Create a collaboration.
PUT --no-wait (LRO)
ParameterRequiredDescription
--resource-groupYesResource group name
--collaboration-nameYesCollaboration name (pattern: ^[a-zA-Z0-9][-_a-zA-Z0-9]{0,61}[a-zA-Z0-9]$)
--locationYesAzure region
--consortium-typeYese.g. ConfidentialACI
--user-identityYesJSON: {tenant-id, object-id, account-type}
--tagsNoTags key=value
--kindNoResource kind

Example:

az managedcleanroom collaboration create \
  --resource-group testrg \
  --collaboration-name ContosoCollaboration \
  --location northeurope \
  --consortium-type ConfidentialACI \
  --user-identity "{tenant-id:fd3c3665-...,object-id:fd3c3665-...,account-type:microsoft}"
